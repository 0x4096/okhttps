# OkHttps Gson

#### 介绍

基于 OkHttps 与 Xml 集成扩展（尚未发布）


##### Maven

```xml
<dependency>
     <groupId>com.ejlchina</groupId>
     <artifactId>okhttps-xml</artifactId>
     <version>2.4.1</version>
</dependency>
```

##### Gradle

```groovy
implementation 'com.ejlchina:okhttps-xml:2.4.1'
```
